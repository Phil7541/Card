# Personal Portfolio Website

A responsive personal portfolio website built to showcase my software and embedded systems projects, technical skills, and ongoing development work. The site was designed as both a professional portfolio and a practical front-end project to improve responsive design, accessibility, and UI development skills.

## Features

* Responsive single-page portfolio layout
* Dynamic project rendering using Vue.js
* Animated navigation bar based on scroll position
* Accessible “Skip to Content” navigation
* Custom SVG patterned header background
* Interactive contact buttons with hover effects
* Mobile-first responsive design using CSS variables and `clamp()`
* Project cards with support for future screenshots and repository links

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio-website.git
```

2. Open the project folder:

```bash
cd portfolio-website
```

3. Launch with a local development server.

Example using Python:

```bash
python -m http.server
```

4. Open the displayed localhost address in a browser.

## Design Goals

The website was intentionally designed to feel personal while remaining clean and professional. The visual style combines bold colours, large typography, and subtle programming-inspired background elements to reflect an interest in embedded systems and low-level development.

A major focus of the project was creating a layout that scales cleanly across a wide range of screen sizes without relying on external CSS frameworks.

---

## What I Learned

Through this project I improved my understanding of:

* Responsive layout design using modern CSS techniques
* Structuring reusable front-end components with Vue.js
* Managing dynamic UI behaviour using the Intersection Observer API
* Designing accessible navigation and page structure
* Creating scalable design systems using CSS custom properties
* Building cleaner and more maintainable front-end codebases

One particularly interesting challenge was balancing large desktop typography with mobile responsiveness while keeping the design visually consistent across breakpoints.
